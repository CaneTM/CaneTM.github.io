Download Instructions

1) Select Game Build
2) Click on the Link/Button to Download File
3) Open File and Extract Using Zip Opener
4) Open Uncompressed file, and Open “KnightQuest.exe” (for Windows OS)
5) Enjoy!

Gameplay Instructions

Keybinds:

1) A - Move Left
2) D - Move Right
3) Space Bar - Jump
4) Tabs - Pause Menu


Gameplay

The overall objective of the game is to pass each level by collecting coins and defeating enemies

*Note: In the current build (0.0.3) the player is only able to play on 1 map/level and no enemies have been input as of date. Please check in for updates and new releases of KnightQuest.
